# BalmJS template - legacy

> A jQuery project for older browsers like Internet Explorer 6-8, Opera 12.1x or Safari 5.1+

## Installation

```sh
npm i -g balm-cli
balm init legacy my-project

cd my-project
yarn
# or
npm i
```

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```

> thx [BalmJS](https://balmjs.com/)
